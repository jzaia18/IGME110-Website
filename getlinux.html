Get Linux

Ubuntu

Ubuntu is a fantastic operating system for those looking to get into Linux for the first time. It is user friendly, fast, and has a great community that is very welcoming to anyone who wants to join. Furthermore, it is Steam's endorsed version of Linux. It is maintained by Canonical Ltd., a for-profit corporation which supports its ongoing development. Find out more about Ubuntu on Ubuntu's official website.

Arch Linux

Arch Linux is my personal favorite distribution of Linux. It is minimalistic, flexible, and very fast. It's greatest advantage over other Linux distributions is the AUR, which allows for easier installs of 3rd party software. While Arch is fantastic for power users, it is certainly not for beginners. If you are new to Linux and would like to learn more about how it works, try a distribution such as Ubuntu first and see how you like it. Arch is not backed by any corporation, but is rather developed by a community without specific versions or releases (rolling release). Learn more about arch on their official website.
