Get Linux

Ubuntu

Ubuntu is a fantastic operating system for those looking to get into Linux for the first time. It is user friendly, fast, and has a great community that is very welcoming to anyone who wants to join. Furthermore, it is Steam's endorsed version of Linux. Find out more about Ubuntu on Ubuntu's official website.
